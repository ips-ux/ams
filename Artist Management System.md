Artist Management System:

Design ideas.

Language: TypeScript

Frontend: React.js (with Tailwind CSS for the custom color templates "Light, Dark, Live (default ableton colorway), Loops (default FL studio colorway), Logical (Default Logic colorway), Tools (default Avid Pro Tools colorway), Cubes (default Steinberg Cubase colorway)" - The app must be both mobile and desktop/laptop friendly.

State Management: TanStack Query (to keep artist data in sync)

Desktop Wrapper: Tauri (for a lightweight, professional feel)

The Aspire Project Tracking System helps you guide a track from the initial idea all the way through to release and provides you a place to store the relevant information, mix versions, creative assets and a method to track submissions and releases.
When starting a new song, add +New and fill out details in the properties section. (This feature shoudl auto trigger from export folder/subfolder monitoring if enabled by the user)
As you progress the song through the 5-step system and onto Mixing and Mastering, update the Project Status property or drag & drop into the new column.
-Spark
-Expand
-Arrange
-Elevate
-Finalize
-Mixing
-Mastering
-Complete > Push to track submission workflow
When reviewing mix versions, you can preview the music file on the song’s card and then create notes specific to that version.
When the song is complete and ready for release, mark it as “Complete” and it will now appear in the “Submissions” section.
After submitting the track to target labels, or deciding to self-release, move the track block to the appropriate column.
Once a release date is set, mark the due date and release date properties in the song block and move it to the “Scheduled” column. This will automatically appear in your calendar if it is set up correctly.
Once the track is released, mark it as “Released” and it will now appear in your discography.
Finish filling out all of the property fields for the track by including links to streaming, final creative assets, credits, and any other relevant information.

***Welcome to your Artist Management System!** If you’re new to Notion, don’t worry – it’s a powerful tool that’s quite user-friendly once you get the hang of it. Follow these steps to start using the music production template I’ve provided:*

### Step 2: Access the Template

1. **Template Link**: Open the link to the template I’ve shared with you, or click the link below: [https://aspireartistacademy.notion.site/Artist-Management-System-e0c78b21f52f4e6a9bab9d49594c5168](https://www.notion.so/Artist-Management-System-Full-e0c78b21f52f4e6a9bab9d49594c5168?pvs=21)

1. **Home Page**: The home page of the template serves as your dashboard. Here you will see links to different sections like “Vision,” “Music Projects,” and more.
2. **Sidebar**: Use the sidebar on the left to quickly navigate between different pages and databases within the template.
